Vim 系列（一） Vim tutor 要点小结

Published: 周二 27 一月 2015
By Charles Chen

In misc.

tags: 工具

因为各种原因，准备慢慢把主要的编辑器从 Sublime Text 3 迁移至古老的 Vim。今天开始，不定期地更新一系列的学习笔记。这篇是第一篇，大致整理了一下自带的教程（vim tutor）中的主要内容。

启动 Vim。把 vim 的目录添加到 PATH 中，则输入 vim 可直接在命令行中启动 Vim，而 gvim 则启动带界面的版本。

Vim 有多种模式，其中最基本的是「普通模式」，启动 Vim 后默认打开的就是普通模式，普通模式中光标是覆盖在字符上的方块，而如果在「插入模式」中，光标是字符之间的一条闪烁的线（绝大多数程序中也如此）。如果处在其它模式中，多按几次 ESC 就可以回到普通模式。

普通模式下的光标移动。最基本的是用 hjkl 来实现上下左右，这一点刚上手，估计需要一段时间来适应。其它常用的，` 是移动到行首，$是移动到行尾，分别相当于键盘上的 HOME 和 END。w移动到下一单词的第一个字符，e移动到当前单词的最后一个字符，再按则跳到下一个单词的词尾。%跳转到配对的括号。输入行号紧接着大写 G 直接跳到指定的行（的开头），只输入 G 跳到最后一行，gg` 跳至第一行。

进入「插入模式」的命令。i 是在光标前插入文本，a 是在光标后，A 是在行末。o 是在当前行下方插入新行，O 是在当前行上方。

删除文本。x 删除光标所在字符。dd 删除整行。删除命令可以与移动命令结合，比如 dw 是当前光标删除至下一单词，d$ 是当前光标删除至行尾。

撤消：u 撤消一步的操作，U 撤消对整行的改动，ctrl-R 撤消撤消的操作本身（相当于一般程序中的 ctrl-Y）。

退出 Vim：:q! 放弃修改并退出，:wq 保存并退出。

执行外部命令。:!command 可以执行外部的 command 命令，如 :!dir、:!del filename。

命令自动补全。输入 : 开头的命令时，按 ctrl-D 可以显示所有的命令提示，按 TAB 可以自动补全。

帮助。输入 :help cmd 可以找到关于 cmd 命令的帮助，再输入 :q 可以关闭帮助窗口。

PS. 本文的内容并不涵盖教程中的全部，只有一些必须的基础，比如还没有讲到如何编辑 vimrc 文件以保存自定义设置、如何搜索替换等等。后面再继续更新。
